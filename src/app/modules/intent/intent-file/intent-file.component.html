Importer / exporter des connaissances

Exporter les connaissances

Exporter les connaissances vous permet d'avoir un fichier excel rempli. Il contiendra toutes les modifications apportées via le back office depuis sa date d'importation.

Importer des nouvelles connaissances

Importer un nouveau fichier de connaissances permet de mettre à jour celui actuellement en place. Dans ce cas, les connaissances remplacées sont automatiquement archivées dans l'historique.

warning Pour une nouvelle connaissance, l'identifiant doit aussi être nouveau sur le fichier excel.
